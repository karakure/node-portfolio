# Node Auth Portfolio

## 概要
Node.js + Express による Session 認証サンプルです。

## 技術スタック
- Node.js 20
- Express
- EJS
- express-session
- Docker

## セットアップ
```bash
npm install
npm run dev


## Lint 設定
ESLint（Flat Config）を採用し、
Node.js / TypeScript それぞれに適した設定を分離して管理しています。



## Requirements
- Docker Desktop (Windows / macOS)
- Docker Engine (Linux)

## Setup
docker compose up





① PC起動
② Docker Desktop 起動（常駐）
③ VSCode 起動
④ docker compose up
⑤ コーディング
⑥ Ctrl + C で停止
